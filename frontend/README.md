
# Employee Frontend (Vite + React)

## Overview
Simple React app that connects to the GraphQL backend and demonstrates:
- Hamburger menu (single-level submenu)
- Horizontal menu
- Grid view (10 columns example)
- Tile view with bun actions (edit, flag, delete)
- Modal/expanded view for full record details
- Pagination, search, optimistic UI for flag/delete

## Run
1. cd frontend
2. npm install
3. create .env with VITE_GRAPHQL_URL if backend is not on default
4. npm run dev (default port 5173)
